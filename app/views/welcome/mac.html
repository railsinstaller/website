RailsInstaller

Get up & running with Rails

  • Step 2: Instructions for MAC (Click for Windows)


    Now that you've downloaded the kit let's get started!

    1. Open the downloaded TAR file to begin the installation and follow the onscreen instructions to setup GIT. This process takes about 20 minutes so I would go grab a cup of coffee and come back!

    2. Once installation is complete go ahead and open up Terminal under /Applications/Utilities

      3. Next we will run a few commands to make sure everything was setup correctly:

       $ git --version

      This will display what version of GIT you have.

       $ rvm info

      This will display information about your Ruby Version Manager environment.

       $ ruby -v

      This will display information about your Ruby version to ensure everything was setup properly.

       $ rails -v

      This will display information about your Rails version to ensure everything was setup properly.

    3. Now that we've ensured everything is setup correctly we can now create a brand new rails application.

      4.  $ cd Desktop/ 

       $ rails new railsinstaller_osx_demo

    4. Once that's complete we can visit the application folder and start Rails server to see the site in our browser.

      5.  $ cd railsinstaller_osx_demo/ 

       $ rails server

      Visit http://localhost:3000 in your web browser to see the default Rails welcome page.

    5. The remaining steps will require you to have a text editor. There are various text editors you can use out there but I'll put one below that is fairly popular in the developer community.

      6. - Visit link to download: Sublime Text

      - After downloading and installing Sublime Text you'll need to install its command line functionality:

      • The first task is to make a symlink to subl. Assuming you've placed Sublime Text 2 in the Applications folder, and that you have a ~/bin directory in your path, you can run: 

        •  $ ln -s "/Applications/Sublime Text 2.app/Contents/SharedSupport/bin/subl" ~/bin/subl

      • You can now visit the folder of your rails application and open your new rails app with Sublime Text

        •  $ cd railsinstaller_osx_demo 

         $ subl .

    6. Now that you have your rails demo app open in the text editor edit the /config/routes.rb file using the navigation panel. Once you're there uncomment the following line of code by removing the "#" sign in front of it. Be sure to to SAVE that file after the change

      7.  root :to => 'welcome#index'

    7. Next you'll need to visit the command line prompt and create a welcome controller for the index action.

      8.  $ rails g controller welcome index

    8. Now lets edit the welcome index.html which can be found under app/views/welcome/index.html.erb, and change the content to say something like the following:

      9.  <h1>Welcome to the Rails Installer Demo!!!</h1>

      Remember to SAVE that file after change.

      If you restart Rails server and visit http://localhost:3000 you'll see the exact same page because Rails is still serving your index.html file in the public folder. You'll need to remove the file at /public/index.html within Sublime Text. Then when you refresh the browser you should see your new page!

    9. Now for the FINAL stretch, using version control and committing your changes to Github. First let's visit your Github account at https://github.com/username, sign up if you don't have one.

    10. Next we'll need to make sure your SSH key is all setup. On your Github page go to Account Settings then SSH Keys on the left hand side and click on Add SSH Key. You can name your key whatever you want. Now follow the command line steps below to generate your SSH Key

      11.  $ ssh-keygen -t rsa -C "youremail@blank.com"

      Be sure to enter a nice strong password when prompted

       $ pbcopy < ~/.ssh/id_rsa.pub

      Now go back to Github and paste the copied key into the required section. Save it.<>/p>

    11. Next we'll create a new repository on Github, you can call it whatever you want but it's best to keep it similar to our app titled "railsinstaller_osx_demo"

    12. Finally we'll finish it off with the following commands to push your code to Git and Github.

      13. Command Line:

      •  $ git init 


      •  $ git add . 


      •  $ git commit -m "first commit" 


      •  $ git remote add origin git@github.com:accountname/railsinstaller_osx_demo.git 


      •  $ git push origin master

        • Enter your Github.com credentials when prompted then press ENTER.


    It's OFFICIAL, You've created your first Rails Project!

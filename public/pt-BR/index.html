Ao trabalho.

O que é?

RailsInstaller é a maneira mais rápida para desenvolver aplicações com Ruby on Rails do zero. Se você tem Windows ou Mac, RailsInstaller te dá cobertura.

Pra quem é?

Você algum dia já iniciou um novo trabalho com um computador novo? Configurando tudo para poder começar a produzir pode demorar horas. O RailsInstaller agiliza o processo para desenvolvedores Rails para permitir o seu sucesso.

Please do not run the installer on OSX Mavericks until further notice.

E agora

Após instalar o kit, dê uma olhada no Rails Guides para informações sobre desenvolvimento de aplicações com Ruby on Rails.

Um outro grande recurso é o Tutorial de Ruby on Rails por Michael Hartl. Mesmo o tutorial sendo focado em usuários de Mac, ele também disponibiliza grande valor e conhecimento para usuários de Windows. Além disso, são aproximadamente 15 horas de vídeo acompanhadas caso seja melhor algo mais visualmente orientado para você.

Se você precisa de ajuda com o RailsInstaller, por favor veja nosso Google Group onde você pode ver posts de outros usuários assim como fazer suas perguntas. Quaisquer problemas com o instalador podem ser reportados no Github nos respectivos repositórios ( railsinstaller-windows ou railsinstaller-nix ). Há também a sala #railsinstaller no Freenode onde você pode conseguir ajuda num chat em tempo real
